# Screenshots

Place UI screenshots here for documentation and README embeds.

## Naming Convention

```
hero-section.png
chat-demo.png
features-section.png
mobile-view.png
lead-capture-flow.png
light-mode.png
dark-mode.png
```

## When to Add Screenshots

- ✅ After Stage 2 CSS is confirmed working in browser
- ✅ After Stage 3 JavaScript — capture the live chat demo
- ✅ After lead capture flow is complete
- ✅ One mobile screenshot (Chrome DevTools → responsive mode)
- ✅ One light mode screenshot (once theme toggle is built)

## How to Take Good Screenshots

1. Set browser window to exactly **1280 × 800** for consistency
2. Use Chrome DevTools device toolbar for mobile view (**390 × 844**)
3. Crop tightly — no browser chrome (tabs, address bar) unless intentional
4. Save as `.png` for best quality
